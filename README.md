# School Management System

## Vision
A comprehensive, user-friendly School Management System designed to streamline administrative tasks, enhance teacher productivity, and provide transparency for students and parents.

## Tech Stack
- **Frontend:** React + Tailwind CSS
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Authentication:** JWT

## Structure
- `/frontend`: React application (Vite)
- `/backend`: Node.js API server
